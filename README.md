# 👻 GhostPad

Полупрозрачный блокнот, **невидимый при демонстрации экрана**.

---

## Возможности

| Функция | Описание |
|---|---|
| 🔒 Защита от захвата | Не виден в OBS, Zoom, Teams, Google Meet |
| 👻 Полупрозрачность | Стеклянный UI поверх любых окон |
| 🎚️ Регулировка прозрачности | Слайдер в правом верхнем углу, значение запоминается |
| 🗂️ Папки | Иерархическое дерево заметок: создание/переименование/удаление папок, вложенность, drag-and-drop заметок в папки |
| 🎨 Оформление | Цвет фона, акцента и тинта + пресеты (Ghost Dark, Cyberpunk, Matrix, Frosted Glass) |
| 🪟 Управление окном | Свернуть, развернуть/восстановить, полноэкранный режим (F11); размер и позиция запоминаются |
| ✏️ Редактор | Жирный, курсив, подчёркивание, зачёркивание |
| 🎨 Цвет шрифта | 7 быстрых цветов + свой через палитру |
| 📝 Несколько заметок | Список заметок с автосохранением |
| 🖱️ Сквозной режим | Кликаете сквозь блокнот на экран за ним |
| 🙈 Скрыть/Показать | Мгновенное скрытие горячей клавишей |
| 💾 Сохранение настроек | Прозрачность, тема, состояние окна и дерева папок переживают перезапуск |

---

## Горячие клавиши

| Клавиша | Действие |
|---|---|
| `Ctrl/⌘ + Shift + T` | Включить/выключить **сквозной режим** |
| `Ctrl/⌘ + Shift + H` | **Скрыть/показать** окно |
| `F11` | Полноэкранный режим (вкл/выкл) |
| `Esc` | Закрыть открытую панель / выйти из полноэкранного режима |
| `Ctrl/⌘ + B` | Жирный |
| `Ctrl/⌘ + I` | Курсив |
| `Ctrl/⌘ + U` | Подчёркнутый |

## Установка и запуск

### Требования
- [Node.js](https://nodejs.org/) v18 или новее

### Шаги

```bash
# 1. Распакуйте архив и перейдите в папку
cd ghost-pad

# 2. Установите зависимости
npm install

# 3. Запустите
npm start
```

## Как это работает

- **`setContentProtection(true)`** — системный API Windows и macOS, который запрещает любому ПО (Zoom, OBS, Teams…) захватывать содержимое окна. На Linux не поддерживается.
- **`setIgnoreMouseEvents(true, { forward: true })`** — сквозной режим: клики проходят насквозь, но система по-прежнему получает события мыши для фона.
- **`backdrop-filter: blur`** + прозрачный фон окна — стеклянный эффект без потери читаемости.
- **Прозрачность** регулируется альфой CSS-фона стекла (а не `BrowserWindow.setOpacity`), поэтому текст всегда читаем и защита от захвата / сквозной режим / always-on-top не затрагиваются.
- **Хранение:** заметки и папки — в `userData/ghostpad-notes.json` (формат обратно совместим со старым массивом заметок); настройки (прозрачность, тема, размер/позиция окна, раскрытые папки) — в `userData/ghostpad-settings.json`.
